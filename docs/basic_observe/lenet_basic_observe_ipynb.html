Firstly, lets observe feature map shape by simply call get_activation_info with method = Shape! Also, we can observe weights in same way. get_weight_info does not require inference. Observer methods are defined in mrfi.observer, you can also add custom observer by mrfi.add_function

Next, let's simply draw variable distribution figure by UniformSampling observer! This method defaults to uniformly sampling up to 10000 data points, so you don't have to worry about memory overflow due to too large tensors.

In [9]:
act_samping = get_activation_info(fi_model, input_images, 'UniformSampling', module_name = ['conv', 'fc'])
fig, axs = plt.subplots(2,3)
for i, (name, value) in enumerate(act_samping.items()):
    ax = axs[i//3, i%3]
    ax.hist(value, bins=20)
    ax.set_title(name)
axs[1,2].violinplot(list(act_samping.values()))
