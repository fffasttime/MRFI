observeFI_experiment(fi_model, input_data, use_golden=True, observers_dict=None) ¶

Run fault inject experiment and return internal oberver results.

Parameters:

Name Type Description Default
fi_model MRFI

Target model.

 required
input_data Union[torch.Tensor, DataLoader]

A input tensor or a dataloader.

 required
use_golden bool

If True, run both golden run and fault injection per batch, most of fault injection observer requires golden run before fault inject.

 True
observers_dict Optional[dict]

A dict of observer config handlers to get result. if None, return all observer result.

 None

Returns:

Type Description
dict

Dictionary of observer result, keys are observer name and values are result of observer.result().
